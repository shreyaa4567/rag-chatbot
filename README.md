# RAG Chatbot — Website-Aware Question Answering System

A Retrieval-Augmented Generation (RAG) chatbot that answers questions based solely on a target website's content. Built with Ollama, Qwen3, ChromaDB, and Python.

## Tech Stack

| Component | Technology |
|---|---|
| LLM | Qwen3:8b via Ollama |
| Embeddings | nomic-embed-text via Ollama |
| Vector Store | ChromaDB |
| Web Crawler | BeautifulSoup + requests + tldextract |
| RAG Framework | LangChain |
| API | Flask |
| Frontend | ASPX / .NET / IIS |
